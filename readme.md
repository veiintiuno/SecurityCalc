# SecureCalc

Calculadora modular en C.

## Compilar
make

## Ejecutar  
./securecalc

## Características
- 4 operaciones básicas
- Manejo de decimales
- Validación de entrada
- Código modular (headers + source)

## Estructura
inc/    - Headers (.h)
src/    - Código fuente (.c)
Makefile - Compilación automática
